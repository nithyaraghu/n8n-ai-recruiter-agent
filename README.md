This agent automates the resume screening process by:
1. **Ingesting Resumes:** Accepts PDF resumes via Chat Upload or Google Drive.
2. **Dynamic Analysis:** Compares the resume against a *dynamic* Job Description provided by the recruiter.
3. **AI Scoring:** Uses LLMs to calculate a 0-100 match score and identify missing skills.
4. **Automated Actions:**
   - **> 70 Score:** Drafts an interview invitation email.
   - **< 70 Score:** Logs the candidate in Google Sheets and sends a polite rejection email with *specific, constructive feedback* generated by AI.
